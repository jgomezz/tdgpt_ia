
# Configurar Virtual Environment en Python

## 1. Pasos de instalación - **En macOS:**
--- 

### 1.1. Crear el virtual environment
```bash
brew install python@3.12 
python3.12 -m venv .venv
```

### 1.2. Activar el virtual environment
```bash
source .venv/bin/activate
```

### 1.3. Verificar la activación
```bash
which python 
```

### 1.4. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 1.5. Desactivar el virtual environment
```bash
deactivate
```
